Glossary of Terms

Certifications to consider for a career in cybersecurity

Access Control List (ACL):
A set of rules used to control network traffic and reduce network attacks by specifying which users or systems can access certain resources.
Bandwidth:
The maximum rate of data transfer across a network path, typically measured in bits per second (bps).
Cloud Computing:
The delivery of computing services (servers, storage, databases, networking, software) over the Internet (“the cloud”) to offer faster innovation, flexible resources, and economies of scale.
Cryptography:
The practice and study of techniques for securing communication and information through the use of codes.
Denial of Service (DoS):
An attack that aims to make a machine or network resource unavailable to its intended users by overwhelming it with a flood of illegitimate requests.
Encryption:
The process of converting information or data into a code to prevent unauthorized access.
Firewall:
A network security device that monitors and controls incoming and outgoing network traffic based on predetermined security rules.
IP Address:
A unique identifier for a device on a network, used for communication between devices.
Intrusion Detection System (IDS):
A device or software application that monitors a network or systems for malicious activity or policy violations.
Network Protocol:
A set of rules governing the format of messages exchanged between computers on a network.
Network Topology:
The arrangement of different elements (links, nodes, etc.) in a computer network.
Packet Sniffer:
A tool used to capture and analyze network traffic.
Phishing:
A method of trying to gather personal information using deceptive e-mails and websites.
Router:
A device that forwards data packets between computer networks, directing traffic on the Internet.
Subnet:
A logical subdivision of an IP network, allowing for better management of the network.
Vulnerability:
A weakness in a system that can be exploited by attackers to gain unauthorized access or perform unauthorized actions.
Wi-Fi:
A technology that allows electronic devices to connect to a wireless LAN (Local Area Network), typically using the 2.4 GHz or 5 GHz radio bands.
